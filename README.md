## Navigation

* **F1** — open / hide menu  
* **F2** — switch presets  

> ⚠️ **Important:** `F1` and `F2` only work in a **dual‑PC** setup.

## After Updating

1. Check which preset is active.  
2. Select your monitor if you use multiple displays.  
3. If something breaks after a game update: update `offsets.txt`, enable reading from it, then press **Restart**.

Feedback and suggestions are welcome.

## Obtaining DLLs

All required files (`vmm.dll`, `leechcore.dll`, `FTD3XX.dll`) are included in the official **MemProcFS** releases.

1. Go to [github.com/ufrisk/MemProcFS/releases](https://github.com/ufrisk/MemProcFS/releases)  
2. Download the latest `MemProcFS-vX.X.zip`  
3. Extract the archive – you will find all three `.dll` files (root or `bin` folder)  
4. Copy them to:  
   `C:\Users\2PC\Desktop\WTC\lib\dma`
