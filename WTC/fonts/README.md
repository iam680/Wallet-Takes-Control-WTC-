# Fonts Directory

## Required Fonts

Place the following font files in this directory:

1. **arialbd.ttf** - Arial Bold (required for ESP text rendering)
   - Download from: Windows system fonts (`C:\Windows\Fonts\arialbd.ttf`)
   - Or download from any legitimate font source

2. **arial.ttf** - Arial Regular (optional, can be used as alternative)
   - Download from: Windows system fonts (`C:\Windows\Fonts\arial.ttf`)

## How to Get Fonts

### Option 1: From Windows System
Copy `arialbd.ttf` from `C:\Windows\Fonts\` to this directory.

### Option 2: Alternative Fonts
If you prefer other fonts with Cyrillic support, you can use:
- `Roboto-Bold.ttf`
- `Consolas.ttf`
- Any TTF font with Cyrillic character support

## Note
The cheat currently uses `arialbd.ttf` (Arial Bold) instead of the Korean font `NotoSansKR.ttf` to properly display Cyrillic characters and avoid encoding issues.
